# Internetworking Protocol 
## Assignment 1: Socket

Implement Client and Server:

### Client
- Input 2 integers a and b
- Send a, b to the server

### Server: 
- Receive a and b
- Send a + b, a - b, a * b, a / b, a % b to the client

### Client:
- Receive numbers from the server
- Print all the received numbers
